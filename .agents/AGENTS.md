# Workspace Rules

### Premium UI/UX Enforcement
Whenever a task involves creating, refactoring, or reviewing UI components (frontend):
1. **Always** adhere to the guidelines in `core-ui-principles`, `ui-scaling-mastery`, and `react-bits`.
2. **Always** generate or consult the design system using the `ui-ux-pro-max` skill before making visual changes.
3. Prioritize fluid layouts, responsive scaling (`clamp`), and high-quality micro-interactions (150-300ms) over static, hardcoded designs.
